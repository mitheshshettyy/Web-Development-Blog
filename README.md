# CODE.it: Web Development Essentials

## ðŸ’» Project Description

**CODE.it** is a responsive, single-page informational website designed to introduce users to the core essentials of **Web Development**. It provides a clear, categorized overview of foundational languages (HTML, CSS, JavaScript) and backend technologies (PHP, Node.js), along with key concepts like Databases, APIs, and Full-Stack Frameworks.

The site is built with a focus on a modern design, providing a seamless viewing experience on any device.

## âœ¨ Key Features

* **Responsive Design:** Built with **Bootstrap 4** to ensure the layout is optimized for desktops, tablets, and mobile devices.
* **Dark/Light Mode Toggle:** A user-friendly feature allowing visitors to switch between a comfortable dark theme and a standard light theme.
* **Video Tutorials:** Dedicated sections for HTML, CSS, JavaScript, PHP, and Node.js, each featuring:
    * A **"View on Site"** button that utilizes a custom JavaScript modal to embed the YouTube video directly on the page.
    * An **"Open in New Tab"** button for direct YouTube access.
* **Comprehensive Content:** Structured explanations of:
    * Front-end vs. Back-end development.
    * Core web development concepts (Databases, APIs, Version Control).
* **Team Section:** An organized introduction to the project creators, complete with linked social profiles.

## ðŸ› ï¸ Tech Stack

This project is built entirely using standard client-side web technologies:

* **HTML5:** Provides the semantic structure and content layout.
* **CSS3:** Used for all custom styling, typography, and the dark mode theme.
* **JavaScript (ES6+):** Implements dynamic features, including the Dark Mode toggle and the video embedding pop-up logic.
* **Bootstrap 4:** Utilized for the responsive grid system, navigation, and component styling.
* **Font Awesome:** Integrated for a variety of social and functional icons.
